ğŸ”¹ Linear Search (Basic Idea)

In linear search:

Start from the first element

Compare each element with the target

Stop when the element is found or the array ends

This method is called linear because elements are checked one by one.

ğŸ”¹ Linear Search Code
arr = [6, 3, 9, 2, 7]
target = 9

found = False

for i in range(len(arr)):
    if arr[i] == target:
        print("Found at index", i)
        found = True
        break

if not found:
    print("Not found")

ğŸ”¹ Why We Use break

Once the element is found:

There is no need to continue searching

break stops the loop immediately

This avoids unnecessary comparisons.

ğŸ”¹ Correctness Explanation

Why this works:
This works because each element of the array is compared with the target, so if the target exists, it will be found.
