ðŸ”¹ What is Frequency Counting?

Frequency counting means counting how many times each element appears in an array.

Example:

Array: [1, 2, 2, 3, 1, 1]
Frequencies:
1 â†’ 3 times
2 â†’ 2 times
3 â†’ 1 time


This technique is useful when:

Finding duplicates

Counting occurrences

Solving grouping problems

ðŸ”¹ Basic Approach (Using a Dictionary)

We use a dictionary because:

Keys store the element

Values store the count

ðŸ”¹ Frequency Counting Logic

Steps:

Start with an empty dictionary

Traverse the array

If element exists â†’ increase count

If not â†’ add it with count 1

ðŸ”¹ Code Example
arr = [1, 2, 2, 3, 1, 1]

freq = {}

for x in arr:
    if x in freq:
        freq[x] += 1
    else:
        freq[x] = 1

print(freq)


Output:

{1: 3, 2: 2, 3: 1}

ðŸ”¹ Correctness Explanation

Why this works:
This works because every element is processed once and its count is updated correctly in the dictionary.
